## 9. Портфолио — не работает фильтрация по типам работ
- **Проблема:** Кнопки тегов были без JS-обработчиков
- **Решение:** Создан клиентский компонент `PortfolioFilter` с `useState` фильтрацией. Активный тег подсвечивается, карточки фильтруются мгновенно
- **Файл:** `src/app/(site)/portfolio/PortfolioFilter.tsx`
- **Статус:** ✅ Сделано

## 10. Портфолио — нет слайдера/просмотра фото на детальной странице
- **Проблема:** На детальной странице работы нет возможности листать фото, нет лайтбокса
- **Решение:** Создан `PortfolioGallery` — лайтбокс с полноэкранным просмотром, стрелки навигации, клавиши (←→, Esc), счётчик "1/4"
- **Файл:** `src/app/(site)/portfolio/[slug]/PortfolioGallery.tsx`
- **Статус:** ✅ Сделано

## 11. Портфолио — заглушки вместо фото на детальных страницах

## 12. Страницы не открываются (Блог, Контакты, О студии)
- **Проблема:** 502 Bad Gateway, страницы не грузятся
- **Причина:** OOM kill при билде на сервере (мало RAM)
- **Решение:** Добавлен swap 2GB на сервере
- **Статус:** ✅ Решено (серверная настройка, не код)
