# Воронка продаж — спецификация

> Элементы воронки реализуются в Спринте 4, но Claude Code должен знать о них с самого начала, чтобы заложить правильную архитектуру.

## 1. Exit-Intent Popup
**Компонент:** `src/components/funnel/ExitIntentPopup.tsx`
**Триггер:** курсор к верху экрана (desktop), кнопка «назад» (mobile)
**Условия:** через 15 сек на странице, 1 раз за сессию, не показывать 72ч (localStorage)
**Не показывать:** если форма уже отправлена

3 варианта (A/B через prop `variant`):
- **discount:** «Скидка 10% при записи сегодня» + промокод EXIT10 + таймер до конца дня + форма (телефон)
- **leadmagnet:** «Заберите чек-лист: 20 пунктов проверки детейлинга» + форма (WhatsApp номер)
- **photo:** «Пришлите фото — рассчитаем стоимость за 15 минут» + кнопка → wa.me с предзаполненным текстом

**Дизайн:** overlay чёрный 60% blur, модалка 480px (90vw mobile), bg-bg-card, border-radius 16px

## 2. Callback Widget
**Компонент:** `src/components/funnel/CallbackWidget.tsx`
**Кнопка:** fixed, right-4, bottom-24 (над floating panel), круг 56px, bg-accent-lime, иконка Phone
**Пульсация:** @keyframes pulse-ring 2s infinite
**Desktop:** + текст «Перезвоним за 28 сек» при hover

**Модалка:** «Перезвоним через 28 секунд» → поле телефон → «Жду звонка»
**После submit:** анимированный таймер 28→0 сек + «Ваш номер принят»
**Нерабочее время (>20:00):** «Перезвоним завтра в 10:00»

## 3. Quiz Calculator (КЛЮЧЕВОЙ)
**Компонент:** `src/components/funnel/QuizCalculator.tsx`
**Где:** на главной (встроен в Hero), на остальных — через кнопку «Рассчитать»

4 шага:
1. **Класс авто** — 4 карточки с иконками: Седан / Кроссовер / Купе / Минивэн
2. **Что хотите?** — чекбоксы: Защита кузова / Полировка / Тонировка / Шумка / Оптика / Химчистка
3. **Когда?** — 3 кнопки: На этой неделе / В этом месяце / Пока присматриваюсь
4. **Телефон** — Input phone + Имя + Кнопка «Получить расчёт за 15 минут»

**UX:** прогресс-бар сверху (шаг N из 4), AnimatePresence slide-left между шагами
**После submit:** «Расчёт будет через 15 мин» + предложение WhatsApp
**Данные → CRM:** класс авто, услуги, срочность, телефон, имя, UTM

## 4. Studio Load Bar
**Компонент:** `src/components/funnel/StudioLoadBar.tsx`
**Где:** страницы услуг, над финальным CTA
**Визуал:** горизонтальный прогресс-бар, заполнен на 75-92%
**Текст:** «Загрузка студии на эту неделю: 87%» + «Осталось 2 свободных слота»
**Цвет бара:** gradient #CCFF00 → #FF4444

## 5. Seasonal Timer
**Компонент:** `src/components/funnel/SeasonalTimer.tsx`
**Где:** под header на всех страницах (или только главная)
**Визуал:** полноширинный баннер, текст акции + обратный отсчёт дд:чч:мм:сс + кнопка
**Данные:** из constants.ts SEASONAL_OFFER (менять каждые 2 мес)

## 6. Social Proof Toast
**Компонент:** `src/components/funnel/SocialProofToast.tsx`
**Где:** глобально (layout.tsx)
**Визуал:** тост внизу слева (desktop), вверху (mobile), slide-in 5 сек → slide-out
**Ротация:** каждые 45 сек, данные из SOCIAL_PROOF_ITEMS
**Не показывать:** если форма уже отправлена
